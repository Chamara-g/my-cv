GeNet: A Python based Web Application for Microarray Gene Expression Analysis (Final Year Project)
python Flask HTML css js SQLite R Google Colaboratory
GeNet is designed carefully targeting a wide range of users. Bio informaticians, researchers, computer scientists, statistical analysts, etc can use the application without prior knowledge in other fields. It provides a hassle-free and interactive user experience that makes the analysis process much simpler and enjoyable. GeNet provides many analysis steps including different techniques for pre-processing, visualization, modeling, prediction, and validation of the results with a user-friendly GUI.
